Introducing RECwerk

RECwerk is a native-first waveform and audio editor for Linux, Windows, macOS, and the web. It keeps the direct editing feel of classic waveform editors while adding practical modern workflows for restoration, recording, and quick destructive editing.

The project builds on the excellent AudioMass engine by Pantelis Kalogiros, then extends it with a desktop shell, native file workflows, a Syntrillium-inspired interface, and release packaging for one-click distribution.

:: Feature List ::
  • Desktop and web builds: Electron for native desktop use, plus a matching static web build.
  • Waveform editing: Fast mono and stereo editing with selection-based destructive processing.
  • Real-time recording: Record directly into RECwerk with monitoring controls.
  • Restoration workflow: Built-in click detection and click removal for vinyl-style cleanup.
  • Real-time FX preview: Preview EQ, dynamics, delay, reverb, chorus, bitcrusher, and more before applying.
  • Multiformat support: WAV, MP3, OGG, FLAC, AIFF import and export support.
  • Classic interface direction: A visual language inspired by Syntrillium-era editors without turning into a museum piece.

Getting Started

Drag an audio file into the workspace or open one from the File menu. Once the waveform is loaded, you can zoom, pan, make a selection, preview effects, and apply destructive edits directly to the selected region or full file.

Recording Audio

Use the Record button or the [R] shortcut to capture audio. The recording modal exposes channel selection, sample rate, and monitoring controls so RECwerk can act as a practical capture and cleanup tool in one window.

Releases

RECwerk is packaged for straightforward distribution: Linux uses AppImage, Windows uses a portable executable, and macOS uses a zip build. The first tagged release is intended to be downloadable directly from the GitHub Releases page without installer-first friction.

The Story & Credits

The soul of RECwerk lies in AudioMass, written by Pantelis Kalogiros in 2018. That project proved how far a lean waveform editor could go without dragging in unnecessary complexity, and RECwerk deliberately keeps that fast, direct spirit.

flarkAUDIO took that base out of the browser-tab-only context and pushed it into a desktop-oriented workflow with native packaging, recording support, restoration tooling, and a stronger editor identity. The goal is not to imitate a full DAW, but to be the fast tool you open when you need to edit, repair, trim, capture, and export audio without ceremony.

Performance Considerations

Working with long waveform buffers in a browser-derived audio engine still has real constraints. Large operations can create pressure on the JavaScript runtime and trigger garbage collection at awkward times.

RECwerk mitigates that by offloading heavier work to background workers where possible and by keeping the editing model focused. The aim is practical responsiveness for real editing sessions rather than checkbox feature inflation.
