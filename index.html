RECwerk · waveform editor · desktop + web

Fast waveform editing without DAW baggage.

RECwerk combines a native desktop release line with a browser editor for quick editing, restoration tests, and waveform work. Use the web build for immediate access, or download the desktop builds for the full workflow.

Open development

RECwerk extends the AudioMass engine into a desktop-oriented tool with native packaging, restoration tooling, and a Syntrillium-inspired interface.

What It Does

  • Waveform editing in mono and stereo
  • Click detection and click removal
  • Fast destructive edits and export
  • Real-time FX preview and restoration tools

Browser Fit

  • Open local files and test the UI quickly
  • Good for previews, playback, and quick edits
  • Not the full replacement for native recording workflows
