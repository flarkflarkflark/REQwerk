Introducing RECwerk

RECwerk is a powerful native desktop application designed for professional audio and waveform editing. It provides a fast, responsive interface inspired by classic Digital Audio Workstations like Syntrillium Cool Edit Pro.

This project is a native evolution of the brilliant AudioMass engine, originally created by Pantelis Kalogiros. While the core engine remains a testament to his web-audio expertise, RECwerk has been modernized and "wrapped" for Linux, Windows, and Mac to provide low-latency audio support and deeper hardware integration.

:: Native Feature List ::
  • Native Audio Engine: High-performance stereo and mono support.
  • Real-time Monitoring: Hear your input during recording with up to 200% volume boost.
  • Hardware Selection: Choose specific Input/Output devices for complex multi-card setups.
  • Multiformat support: Native handling of WAV, MP3, OGG, FLAC, and AIFF.
  • Advanced DSP: Real-time preview for AI Noise Reduction, Paragraphic EQ, Compression, and Reverb.
  • Syntrillium Aesthetic: A classic "Cooler" look-and-feel for the modern era.
  • Non-destructive editing: Full undo/redo history for all operations.

Getting Started

To get started, simply drag and drop an audio file into the workspace. Once the file is loaded, you can view the waveform, zoom in, pan around, and select regions for editing.

Recording Audio

To record audio, press the Recording button or use the [R] key. You can adjust your recording settings (channels, sample rate, monitoring) in the recording modal.

The Story & Credits

The soul of RECwerk lies in the AudioMass project, written by Pantelis Kalogiros in 2018. He shared it as an open-source tool to show how far web-audio technologies could go without complex frameworks. His philosophy was simple: "be fast and get the job done."

The flarkAUDIO team recognized the potential of this engine and decided to take it "out of the browser tab" and onto the desktop. By implementing native wrappers, improving recording capabilities (Stereo/Monitoring), and skinning it with the classic Syntrillium vibe, RECwerk was born—honoring its web-based roots while delivering a true desktop DAW experience.

Performance Considerations (The Technical Rant)

Working with the Web Audio API in a native context brings its own challenges. As the original author noted, iterating over multiple long arrays can eventually trigger garbage collection, leading to "crackling."

In RECwerk, we address this by moving heavy operations—such as filter processing—to background threads (Web Workers). We are also working towards a pure WASM implementation for signal modification to outperform traditional JS processing, ensuring that even large audio files can be handled without freezing the UI.

RECwerk is a free, open-source project. We stand on the shoulders of giants.
